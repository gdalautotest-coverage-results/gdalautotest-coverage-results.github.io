     229             : /* __ieee754_log(x)
     230             :  * Return the logarithm of x
     231             :  *
     232             :  * Method:
     233             :  *   1. Argument Reduction: find k and f such that
     234             :  *                      x = 2^k * (1+f),
     235             :  *         where  sqrt(2)/2 < 1+f < sqrt(2) .
     236             :  *
     237             :  *   2. Approximation of log(1+f).
     238             :  *      Let s = f/(2+f) ; based on log(1+f) = log(1+s) - log(1-s)
     239             :  *               = 2s + 2/3 s**3 + 2/5 s**5 + .....,
     240             :  *               = 2s + s*R
     241             :  *      We use a special Reme algorithm on [0,0.1716] to generate
     242             :  *      a polynomial of degree 14 to approximate R The maximum error
     243             :  *      of this polynomial approximation is bounded by 2**-58.45. In
     244             :  *      other words,
     245             :  *                      2      4      6      8      10      12      14
     246             :  *          R(z) ~ Lg1*s +Lg2*s +Lg3*s +Lg4*s +Lg5*s  +Lg6*s  +Lg7*s
     247             :  *      (the values of Lg1 to Lg7 are listed in the program)
     248             :  *      and
     249             :  *          |      2          14          |     -58.45
     250             :  *          | Lg1*s +...+Lg7*s    -  R(z) | <= 2
     251             :  *          |                             |
     252             :  *      Note that 2s = f - s*f = f - hfsq + s*hfsq, where hfsq = f*f/2.
     253             :  *      In order to guarantee error in log below 1ulp, we compute log
     254             :  *      by
     255             :  *              log(1+f) = f - s*(f - R)        (if f is not too large)
     256             :  *              log(1+f) = f - (hfsq - s*(hfsq+R)).     (better accuracy)
     257             :  *
     258             :  *      3. Finally,  log(x) = k*ln2 + log(1+f).
     259             :  *                          = k*ln2_hi+(f-(hfsq-(s*(hfsq+R)+k*ln2_lo)))

     261             :  *                      ln2_hi + ln2_lo,
     262             :  *         where n*ln2_hi is always exact for |n| < 2000.

     264             :  * Special cases:
     265             :  *      log(x) is NaN with signal if x < 0 (including -INF) ;
     266             :  *      log(+INF) is +INF; log(0) is -INF with signal;
     267             :  *      log(NaN) is that NaN with no signal.
