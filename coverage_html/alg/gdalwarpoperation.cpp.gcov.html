     103             : <h2>Warper Design</h2>
     104             : 
     105             : The overall GDAL high performance image warper is split into a few components.
     106             : 
     107             :  - The transformation between input and output file coordinates is handled
     108             : via GDALTransformerFunc() implementations such as the one returned by
     109             : GDALCreateGenImgProjTransformer().  The transformers are ultimately responsible
     110             : for translating pixel/line locations on the destination image to pixel/line
     111             : locations on the source image.
     112             : 
     113             :  - In order to handle images too large to hold in RAM, the warper needs to
     114             : segment large images.  This is the responsibility of the GDALWarpOperation
     115             : class.  The GDALWarpOperation::ChunkAndWarpImage() invokes
     116             : GDALWarpOperation::WarpRegion() on chunks of output and input image that
     117             : are small enough to hold in the amount of memory allowed by the application.
     118             : This process is described in greater detail in the <b>Image Chunking</b>
     119             : section.
     120             : 
     121             :  - The GDALWarpOperation::WarpRegion() function creates and loads an output
     122             : image buffer, and then calls WarpRegionToBuffer().
     123             : 
     124             :  - GDALWarpOperation::WarpRegionToBuffer() is responsible for loading the
     125             : source imagery corresponding to a particular output region, and generating
     126             : masks and density masks from the source and destination imagery using
     127             : the generator functions found in the GDALWarpOptions structure.  Binds this
     128             : all into an instance of GDALWarpKernel on which the
     129             : GDALWarpKernel::PerformWarp() method is called.
     130             : 
     131             :  - GDALWarpKernel does the actual image warping, but is given an input image
     132             : and an output image to operate on.  The GDALWarpKernel does no IO, and in
     133             : fact knows nothing about GDAL.  It invokes the transformation function to
     134             : get sample locations, builds output values based on the resampling algorithm
     135             : in use.  It also takes any validity and density masks into account during
     136             : this operation.
     137             : 
     138             : <h3>Chunk Size Selection</h3>
     139             : 
     140             : The GDALWarpOptions ChunkAndWarpImage() method is responsible for invoking
     141             : the WarpRegion() method on appropriate sized output chunks such that the
     142             : memory required for the output image buffer, input image buffer and any
     143             : required density and validity buffers is less than or equal to the application
     144             : defined maximum memory available for use.
     145             : 
     146             : It checks the memory required by walking the edges of the output region,
     147             : transforming the locations back into source pixel/line coordinates and
     148             : establishing a bounding rectangle of source imagery that would be required
     149             : for the output area.  This is actually accomplished by the private
     150             : GDALWarpOperation::ComputeSourceWindow() method.
     151             : 
     152             : Then memory requirements are used by totaling the memory required for all
     153             : output bands, input bands, validity masks and density masks.  If this is
     154             : greater than the GDALWarpOptions::dfWarpMemoryLimit then the destination
     155             : region is divided in two (splitting the longest dimension), and
     156             : ChunkAndWarpImage() recursively invoked on each destination subregion.
