      59             :         a) no where clause, the table has a primary key and a GIST index on the raster column.

      63             :

      68             :             ii) If some tiles are not registered as sources, then do a SQL query to fetch their metadata
      69             :             and instantiate them and register them.
      70             :             iii) If some tiles are not cached, then determine if the query window is not too big (w.r.t. GDAL cache),
